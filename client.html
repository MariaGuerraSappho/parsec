Setup Instructions

Step 1: The host computer must be running the main Parsec app (index.html) with BuzzBox connected

Step 2: The host must run the WebSocket server: node server.js

Step 3: Find the host's IP address (e.g., 192.168.1.25) and enter it below

Step 4: Click "Connect to Host" to start remote control
